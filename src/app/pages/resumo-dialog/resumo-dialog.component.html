Resumo do projeto - UniBot

Este projeto propõe a integração de um chatbot e criação de uma API (Application Programming Interface - Interface de Programação de Aplicação) para acessar dados relevantes sobre os alunos na universidade, com o objetivo de melhorar a experiência dos usuários ao acessarem informações sobre sua vida acadêmica.

O chatbot foi desenvolvido utilizando técnicas de PLN (Processamento de Linguagem Natural) e é capaz de interagir com os usuários de forma natural, fornecendo informações sobre sua vida acadêmica.

A integração com a API permite ao chatbot acessar dados em tempo real, como horários de aulas, notas e outras informações relevantes.
